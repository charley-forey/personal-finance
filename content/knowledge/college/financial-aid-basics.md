# Financial Aid Basics

## FAFSA
Free Application for Federal Student Aid determines eligibility for grants, work-study, and federal loans. File annually; opens October 1 for the following academic year.

## Expected Family Contribution (EFC)
Calculated from income, assets, family size, and number in college. Lower EFC = more aid eligibility.

## Types of Aid
- **Grants**: Free money (Pell Grant, state grants).
- **Work-study**: Part-time campus jobs.
- **Loans**: Subsidized (need-based) and unsubsidized federal loans.

## CSS Profile
Some private colleges require the CSS Profile for institutional aid. Considers home equity and non-custodial parent income.

## Strategies
Maximize retirement contributions (often excluded from aid formulas), spend down student assets before parent assets, and apply early.
